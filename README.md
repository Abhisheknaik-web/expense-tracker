# 🧠 PersonaTrack - Advanced Personality & Behavior Analytics

A comprehensive, professional-grade personality tracking system that helps you understand your emotional patterns, behavioral trends, and personal growth over time.

## 🚀 Features

### 📊 **Dashboard Overview**
- Real-time statistics and metrics
- 30-day mood trend visualization
- Personality radar chart
- Recent entries quick view
- AI-powered insights at a glance
- Goal progress tracking

### ✍️ **Advanced Entry System**
- 8 distinct mood options (Ecstatic, Happy, Content, Neutral, Anxious, Sad, Angry, Tired)
- Multi-dimensional tracking:
  - Energy levels (1-10)
  - Productivity scores (1-10)
  - Social interaction levels (1-10)
  - Overall wellbeing (1-10)
- Activity tagging (Work, Exercise, Social, Creative, Learning, Relaxation, Entertainment, Family)
- Detailed notes and custom tags
- Timestamp tracking

### 🎭 **Personality Traits Analysis**
- Big Five personality assessment:
  - Openness to Experience
  - Conscientiousness
  - Extraversion
  - Agreeableness
  - Emotional Stability
- Historical trait tracking
- Visual radar chart representation
- Track personality changes over time

### 🔍 **Behavior Pattern Recognition**
- Time of day analysis (Morning, Afternoon, Evening, Night)
- Day of week patterns
- Activity correlation with wellbeing
- Mood trigger identification
- Positive and negative pattern detection

### 🎯 **Goals & Habits Tracking**
- Create and manage personal goals
- Multiple categories (Health, Career, Relationships, Personal Growth, Financial, Creative)
- Deadline tracking
- Completion status
- Progress visualization

### 💡 **AI-Powered Insights**
- Automatic pattern detection
- Personalized recommendations
- Wellbeing trend analysis
- Activity-mood correlations
- Streak motivation
- Energy level insights

### 📅 **Timeline View**
- Chronological entry browsing
- Filter by time period (Week, Month, Year, All)
- Search functionality across notes, tags, and activities
- Detailed entry cards with all metrics
- Visual timeline markers

### 💾 **Data Management**
- Export to CSV (spreadsheet format)
- Export to JSON (complete backup)
- Import from backup
- Generate comprehensive reports
- All data stored locally (complete privacy)

### 🌙 **Dark Mode**
- Toggle between light and dark themes
- Persistent theme preference
- Eye-friendly for night tracking

## 🎯 How to Use

1. **Open the app** - Double-click `index.html` in any modern browser
2. **Navigate** - Use the sidebar to access different sections
3. **Create Entry** - Click "New Entry" and select your mood
4. **Track Metrics** - Adjust sliders for energy, productivity, and social levels
5. **Add Context** - Select activities, write notes, add tags
6. **Save** - Click "Save Entry" to log everything
7. **Analyze** - Explore Dashboard, Patterns, and Insights for deep analysis
8. **Set Goals** - Create personal development goals
9. **Track Personality** - Complete periodic personality assessments
10. **Export Data** - Backup your data anytime
