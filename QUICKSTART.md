# 🚀 Quick Start Guide - PersonaTrack

## Getting Started in 30 Seconds

1. **Open the App**
   - Double-click `index.html`
   - Or right-click → Open with → Your Browser

2. **Watch the Welcome Animation**
   - Enjoy the 2-second intro screen
   - App loads automatically

3. **Create Your First Entry**
   - Click "New Entry" in sidebar
   - Select your mood emoji
   - Adjust the sliders
   - Add activities and notes
   - Click "Save Entry"

4. **Explore Features**
   - **Dashboard** - Overview of your stats
   - **Personality** - Take the Big Five assessment
   - **Patterns** - Discover behavioral trends
   - **Goals** - Set and track objectives
   - **Insights** - Get AI-powered recommendations
   - **Compare** - Analyze different time periods
   - **Calendar** - Visual mood heatmap
   - **Achievements** - Unlock badges
   - **Timeline** - Browse all entries
   - **Export** - Backup your data

## 🆘 Troubleshooting

**Data not saving?**
- Check browser localStorage is enabled
- Try a different browser
- Export and reimport data

**Charts not showing?**
- Add more entries (need at least 3-5)
- Refresh the page
- Check browser console for errors

**Slow performance?**
- Clear old entries (export first!)
- Use a modern browser
- Close other tabs
